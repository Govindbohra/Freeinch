# FREEINCH

Premium architectural hardware brand website built with Next.js App Router and Tailwind CSS.

## Features

- Mobile-first responsive design
- Story-driven homepage
- Shop, product details, blog, about, contact, checkout pages
- Static product data and reusable UI components
- Subtle motion with Framer Motion

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Start the production server:

```bash
npm start
```

## Deployment

This project is ready for GitHub deployment. Connect your repository to Vercel or another hosting provider and deploy the `main` branch.

## Connect Domain

To connect a GoDaddy domain, point your DNS to your hosting provider and use the provider's domain setup process.
